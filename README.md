🧙 Witch Hangman

Witch Hangman is a modern browser-based word guessing game built with HTML, CSS, and JavaScript. Players must guess hidden words one letter at a time before the witch is completely revealed. The game features multiple categories, difficulty levels, an on-screen keyboard, physical keyboard support, and a responsive design for desktop and mobile devices.

Features
🧙 Animated witch hangman with six body parts
🏷️ Multiple categories (Animals, Food, Places, Games, Disney, Marvel, Movies, Pokémon, Science, Sports, Technology, Fantasy, and more)
🔥 Easy, Medium, and Hard difficulty levels
📝 Displays category, difficulty, and word structure (e.g., 2 (6, 7))
⌨️ Supports both on-screen and physical keyboard input
📱 Fully responsive for desktop, tablet, and mobile
🎲 Random word selection from a database of hundreds of words
🎯 Clean, modern UI with smooth gameplay
